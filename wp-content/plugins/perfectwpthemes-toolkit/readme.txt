== Description ==

An essential toolkit for themes made by perfectwpthemes (https://perfectwpthemes.com/). Perfectwpthemes Toolkit works only with the WordPress themes by perfectwpthemes.com. Perfectwpthemes Toolkit extends functionality for themes made by perfectwpthemes.com.

This toolkit extends features like custom widgets, social sharing, likes, dislikes, theme layouts & one click demo import feature. 


How to use this Perfectwpthemes Toolkit ? 

* Install perfectwpthemes toolkit while installing any themes made by perfectwpthemes
* After installing & activating perfectwpthemes toolkit you will see message to go to welcome page
* You will see necessary actions in recommended actions tab.
* Under Recommended action tab please feel free to activate plugins that you will need.
* If you think you don't need any of them ignore that recommended action.
* Once you are done with recommended action plugins go to Apperence > Import demo content
* Or, in this point you can also click on import demo button
* Choose your demo by checking live preview.
* Keep Perfectwpthemes Toolkit plugin along with you till you are using themes from us.
* Feel free to contact perfectwpthemes support team at ( https://perfectwpthemes.com/support/ ) if you need any assistance.


== Frequently Asked Questions ==

= What is use of Perfectwpthemes Toolkit plugin ? =

Perfectwpthemes Toolkit extends theme functionality & helps users to import demo contents. It also add necessary support & documention information under Apperence > theme about page

= Where is theme about page located ? =

Theme about page is located at Apperence > theme about page

= Where is demo content import located ? =

Demo import page is located at Apperence > import demo content

= How can I get assists if I need any help ? =

You can contact us at https://perfectwpthemes.com/support/

== Installation ==

*** From your WordPress dashboard ***

1. Visit 'Plugins > Add New',
2. Search for 'Perfectwpthemes ToolKit' and install the plugin.
3. Activate 'Perfectwpthemes ToolKit' from your Plugins page.
